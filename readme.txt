Enable instant WooCommerce checkout via WhatsApp with auto-generated invoices for seamless order processing.

== Description ==

📲 **WhatsOrder** is a powerful WooCommerce extension that enables a **fast, seamless checkout via WhatsApp**. Customers can place orders and instantly receive an **auto-generated invoice link** for easy confirmation and payment. This enhances customer engagement, improves conversion rates, and simplifies order processing.

**🔹 Why Use WhatsOrder?**  
✔ **Instant WhatsApp Checkout** – Convert more customers with a quick, chat-based checkout experience.  
✔ **Auto-Generated Invoices** – No need for manual invoices! WhatsOrder automatically generates a downloadable HTML invoice.  
✔ **Seamless WooCommerce Integration** – Works with WooCommerce’s order workflow, stock management, and payment statuses.  
✔ **Customizable Messaging** – Personalize the WhatsApp message, invoice format, and phone number in WooCommerce settings.  
✔ **Lightweight & Fast** – No bulky third-party libraries, ensuring your store remains optimized.  

**Note:** This plugin is not affiliated with or endorsed by WhatsApp or Meta.

== Features ==

 **WhatsApp Checkout** – Customers complete orders via WhatsApp with a pre-filled message.  
 **Instant Invoice Generation** – Generates an HTML invoice link automatically.  
 **WooCommerce Compatibility** – Fully integrates with WooCommerce’s order system.  
 **Customizable** – Modify WhatsApp message templates, invoice colors, and number formats.  
 **No External Dependencies** – No need for PDF libraries or additional plugins.  

== Installation ==

1. **Download & Install**  
   - 📥 Download from [WordPress.org](https://wordpress.org/plugins/whatsorder-instant-checkout-for-woocommerce/) or [yapacdev.com](https://yapacdev.com/).  
   - 🔧 In **WordPress Admin**, go to **Plugins** → **Add New** → **Upload Plugin**, and select the ZIP file.  
   - ✅ Click **Activate** after installation.  

2. **Check Dependencies**  
   - ⚠️ Ensure **WooCommerce** is installed and activated.  

3. **Configure WhatsOrder**  
   - ⚙️ Navigate to **WooCommerce** → **Settings** → **Payments**.  
   - 📞 Find **WhatsOrder – WhatsApp Checkout** and click **Manage**.  
   - ✏️ Enter your WhatsApp number, customize messages, and adjust invoice settings.  

4. **Start Accepting Orders!**  
   - 🛒 Customers can now **select “Pay via WhatsApp”** on the checkout page.  
   - 📄 An invoice link is sent via WhatsApp automatically.  

== Frequently Asked Questions ==

= 🗂️ Does this plugin generate PDF invoices? =  
No, WhatsOrder creates a **lightweight HTML invoice** that customers receive as a download link.  

= 📱 Do I need a WhatsApp Business account? =
No, but using **WhatsApp Business** allows you to leverage additional features like automated replies.  

= 💰 Does the plugin handle payment confirmation? = 
No, orders are marked as **on-hold** until you manually confirm payment.  

= 🆓 Is WhatsOrder free? = 
Yes! It is 100% free and open-source under the **GPL-2.0+ license**.
